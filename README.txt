			Kontrollspørsmål 
Det some implementerer de forskjellige lagene I MVC arkitekturen 
er JavaScript I siste deloppgave er de tre filer, CSS, JS og HTML. 
Model: Html filen, siden det er denne filen som er modellen og holder på dataen vi har. 
View: CSS Fila, denne fila bestemmer hvordan vår data skal vises. Controller: JS er Fila som kan
kontrollere hva som skal vises av informasjonen i HTML fila.
